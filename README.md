# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/9923f79a-7509-44b0-8fe3-5202e994e946

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   `npm install`
2. (Optional) Copy `.env.example` to `.env.local` and fill in values if you use Gemini/API features.
3. Run the app:
   `npm run dev`
4. Open:
   `http://localhost:3000`

## Useful Scripts

- `npm run dev` - start local dev server
- `npm run build` - create production build
- `npm run clean` - remove `dist/` (works on Windows/macOS/Linux)
